# 免責事項・利用範囲

本ソフトウェア（`@shuji-bonji/houki-nta-mcp`、以下「本MCP」）は、国税庁が公開する **通達・質疑応答事例・タックスアンサー**へのアクセスを支援する **個人利用向けのセカンドオピニオン・ツール**として設計されています。

## 通達の法的位置付け

通達は **行政内部文書**であり、国民・裁判所には**直接的な法的拘束力を持ちません**。

最高裁 昭和43.12.24（墓地埋葬法事件）が示した通り、通達は「下級行政機関および職員に対してその職務権限の行使を指揮し、職務に関して命令するために発するもの」です。

| 主体 | 通達の拘束力 |
|---|---|
| 国民・納税者 | ❌ 直接的拘束力なし（参考情報） |
| 裁判所 | ❌ 拘束されない（独自解釈可能） |
| 国税不服審判所 | ⚠️ 参照するが、異なる裁決も可（事前に国税庁長官に意見通知） |
| 税務署員・国税庁職員 | ✅ 職務命令として遵守義務 |

ただし、**税務署員が通達に従って運用するため、実務上は事実上の規範**として機能します。納税者が通達と異なる判断をした場合、税務調査で否認されるリスクが高くなるため、実務では通達を「事実上守るもの」として扱うのが一般的です。

## 質疑応答事例の位置付け

質疑応答事例は「通達を踏まえた個別事例集」で、参考情報としての位置付けです。国税庁自身が「個々の納税者が行う具体的な取引の課税関係は、その取引に係る事実関係等に応じて、この回答の内容と異なることがあります」と注意喚起しています。

## タックスアンサーの位置付け

タックスアンサーは「一般納税者向け簡易解説」で、最も拘束力が低く、参考情報の位置付けです。具体的な納税計画・税務調査対応の根拠にはなりません。

## 参照優先度のまとめ

```
1位: 法律（国会制定）             ── houki-egov-mcp
2位: 政令・省令                   ── houki-egov-mcp
3位: 告示                         ── houki-egov-mcp（一部）
4位: 通達（行政内部文書）         ── 本MCP（houki-nta-mcp）
5位: 質疑応答事例                 ── 本MCP
6位: 文書回答事例                 ── 本MCP（Phase 3 予定）
7位: タックスアンサー             ── 本MCP
```

## 本MCPが返すもの

- 通達・質疑応答事例・タックスアンサーの**事実情報（根拠テキストと出典URL）**
- 法的位置付けの注記（`legal_status` フィールド）

LLM はこの事実情報をもとに、分析・説明・論点整理を行うことができます。それはこのツールの通常の使い方です。

ただし、その結果を**業務上の「正解」として扱うことは本MCPの想定外**です。最終判断は利用者自身、または有資格の専門家（税理士・公認会計士・弁護士）に委ねてください。

## 3層の責任分離

```
┌─────────────────────────────────────┐
│ 利用者   最終判断と責任                │
│  ↑                                  │
│ LLM      事実からの分析・論点整理       │
│  ↑                                  │
│ 本MCP    一次情報の取得・提示のみ       │
└─────────────────────────────────────┘
```

本MCP は最下層の「事実提供」に徹します。中間層の分析は LLM の能力に依拠し、最終判断は利用者（または有資格者）が行います。

## 想定される使い方

- **会計・経理・税務系プロダクト開発**: 消費税区分判定、インボイス対応、電帳法対応の根拠調査
- **フリーランス・個人事業主の実務**: 自分の事業の取扱いを通達レベルで確認
- **個人納税者の自己調査**: 確定申告前の論点整理
- **税理士相談前の準備**: 「この事案は基本通達のどこに該当しそうか」のあたりをつける
- **学習・自習**: 法律→通達→事例の階層的な理解
- **セカンドオピニオン**: 既存の判断（書籍・記事）に対する裏取り

## 想定されていない使い方

- **税理士法第52条**に定める **税務代理・税務書類作成・税務相談**の業としての提供
- **弁護士法第72条**に定める法律事務の提供
- 有償・反復継続的な税務業務の提供
- 税務調査対応の代理人業務

裁判所・税務署・国税不服審判所は「**AI がそう言った**」を根拠として受け付けません。業として提供する場面では、必ず有資格者（税理士・公認会計士・弁護士）の監督下でご利用ください。

## 「個人が自分の案件に使うこと」は業法の規制外

個人が自分自身の案件を調べるために AI を使うこと自体は、税理士法・弁護士法の規制対象ではありません。有償性が無く、他人の業務として提供しているわけではないためです。

ただし、**税務署は「AI の回答」を根拠として扱いません**。本MCP は一次情報の URL を返すことを重視しており、利用者は必要なら自分で国税庁サイトに飛んで確認できます。

## 一次情報の正確性について

- 通達・質疑応答事例・タックスアンサーは頻繁に改正・公開・更新されます。本MCP が返す情報は、取得時点の国税庁サイトの状態を反映します
- 本MCP は国税庁サイトの誤り・遅延・停止・仕様変更について責任を負いません
- 取得した情報は **取得時点の日付と出典URL**を添えて利用することを推奨します
- スクレイピングの仕様変更により取得できなくなる可能性があります

### 国税庁コンテンツの著作権

通達・質疑応答事例・タックスアンサーの著作権は **国（国税庁）**にあります。再配布・改変は **[政府標準利用規約（第2.0版）](https://cio.go.jp/policy-opendata)** の範囲内で可能です。

本MCP は以下のスタンスで国税庁コンテンツを扱います:

1. **取得結果に常に出典 URL を付与** — 一次情報源を必ず示す
2. **取得時刻を付与** — 再現性確保
3. **改変はしない** — テキストはそのまま、Markdown 整形のみ
4. **大量取得は深夜帯** — サイトに負荷をかけない
5. **robots.txt 順守** — 拒否されているパスは叩かない

## 参考: なぜこの立ち位置なのか

- 日本の士業制度（税理士法52条）は「有償」×「個別具体」の二要素が揃った場合に税務業務独占が発動します
- 個人が自分の案件に AI を使うこと自体は規制外ですが、税務署は「AI の回答」を根拠として扱いません
- 本MCP は、個人利用者が AI と協働しつつ **一次情報（通達・事例）へのアクセスを簡単にする**ことで、「叩き台の精度」と「税理士相談コスト低減」の両方に寄与することを目的とします
